# Gabriel Nunes | Portfólio

Portfólio pessoal desenvolvido para apresentar meu perfil, minhas habilidades e meus projetos front-end.

## Visão geral

Este projeto foi construído com foco em:

- identidade visual forte
- layout responsivo
- navegação simples e direta
- apresentação clara de projetos e contatos

## Tecnologias

- HTML
- CSS
- JavaScript

## Estrutura

- `index.html`: página principal do portfólio
- `style.css`: estilos da interface
- `assets/`: imagens utilizadas no site
- `projeto-academia/`: projeto em destaque listado no portfólio
